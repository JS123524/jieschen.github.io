# jieschen.github.io

## Overview

This project is a personal portfolio website developed for the CS 463/563 Intro to Web Development final project. It is designed to present my background, relevant skills, previous experience, selected projects, and contact information in a clear format.

## Features

- Fixed navigation bar with anchor links for quick section navigation
- Responsive layout that adapts to desktop, tablet, and mobile screens
- Project gallery with a Bootstrap image carousel showing simulation progress
- Contact form with JavaScript validation and user feedback
- Accessible design with improved focus styles and color contrast

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap 5

## Project Structure

```text
project-folder/

```

## How to Run

1. Clone or download this repository.
2. Open `index.html` in web browser.
